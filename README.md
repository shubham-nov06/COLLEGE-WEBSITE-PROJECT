# 🧠 DSA-C (Data Structures & Algorithms in C)

![C Language](https://img.shields.io/badge/Language-C-blue?style=flat-square) ![DSA](https://img.shields.io/badge/Topic-DSA-orange?style=flat-square) ![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

This repository contains my **Data Structures & Algorithms in C** learning projects inspired by **Shraddha Didi (Apna College)**.  
It includes arrays, linked lists, stacks, queues, trees, graphs, sorting & searching algorithms implemented in C with clean and commented code.

## 🚀 Features

- Arrays, Strings, Linked Lists (Singly, Doubly, Circular)  
- Stacks & Queues  
- Trees (Binary, BST) & Graphs  
- Recursion & Sorting/Searching Algorithms  
- Practicing problem-solving for competitive programming & interviews
